Javascript?

  • Não é Java
    • Sintaxe parecida
    • Tipagem fraca
    • Sem tipos estáticos
  • Também conhecido como ECMAScript
  • Multiplataforma
  • Multiparadigma
    • Funções de primeira classe
      • Função é um tipo linguagem
      • Orientado a objetos
        • "Prototype-based", ao invés de "class based"1
      • Funcional
      • Imperativo
1 - A versão 6 do ECMAScript adiciona suporte a classes

  • Nomes de variáveis podem iniciar com os caracteres de A até Z (e também a até z), _ (underscore) ou $ (dólar)

Tipos de dados

Tipo Descrição
Boolean Entidade lógica de valores true ou false
Null Representa um valor vazio
Undefined Representa um valor que não foi inicializada
Number Ponto flutuante de precisão dupla. -(253 -1) and 253 -1
String Cadeia de caracters que representa dados textuais
Object Tipo que representa objetos em memória

Number


console.log(undefined + 1);//Resulta na constante NaN
console.log("" + 1);//Resulta na strintg "1"
console.log(42/0);//Resulta na constante Infinity
console.log(42/-0);//Resulta na constante -Infinity
console.log(42/Infinity);//Resulta em 0
console.log(Number.MIN_NUMBER); //Menor valor numérico permitido
console.log(Number.MAX_NUMBER); //Maior valor numérico permitido

String


//Exibe "Hello Javascript"
console.log("Hello".concat(" ").concat("Javascript"));
console.log("Hello Javascript".length) //Exibe o valor 16
console.log("Hello Javascript".substr(0, 5)) //Exibe "Hello"
console.log("Hello Javascript"[6]) //Exibe "J"

Object


var hello = {
  greeting: "Hello",
  name: "Javascript",
  greet: function() {
    console.log(this.greeting.concat(" ").concat(this.name));
  }
}
hello.greet();//Exibirá Hello Javascript
